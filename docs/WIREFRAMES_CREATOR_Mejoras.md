## Resumen de responsabilidades por módulo

```
MÓDULO              PERTENECE A      DESCRIPCIÓN
──────────────────────────────────────────────────────────────────────
Grid de pintura     Cada jugador     Canvas de dibujo individual
PAINT_PALETTE       Cada jugador     Paleta de colores propia
LIMPIAR             Cada jugador     Limpia solo su grid
ALEATORIO           Cada jugador     Randomiza solo su sprite
──────────────────────────────────────────────────────────────────────
DIFICULTAD          Partida          Aplica a los dos jugadores
EJECUTAR_RUN        Partida          Inicia el juego para ambos
──────────────────────────────────────────────────────────────────────
COMUNIDAD           Global           Mismo módulo en single / multi
                                     Clic → tooltip [ P1 ] [ P2 ]
```

---

**Comportamiento al cargar sprite de comunidad en modo multi:**
```
CARGAR_EN_
[ PLAYER_1 ]  [ PLAYER_2 ]
```
El tooltip aparece sobre la card al hacer clic. El jugador elige
a quién se aplica el sprite. El grid del jugador seleccionado
se actualiza inmediatamente con el diseño de la comunidad.
